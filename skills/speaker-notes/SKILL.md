# Speaker Notes

Use this skill when the user needs speaking support rather than only slide copy.

## Goals

Turn slide content into clear spoken delivery that sounds confident, concise, and natural.

## Workflow

### 1. Calibrate delivery context

Identify:

- presenter seniority
- audience familiarity
- speaking time
- tone
- live demo or static deck
- risk of interruption or tough questions

### 2. Build speaking guidance

For each slide, provide:

- speaker goal
- talk track
- transition from previous slide
- emphasis cues
- optional brevity version

### 3. Prepare discussion support

Add when relevant:

- likely objections
- likely executive questions
- concise answers
- fallback wording for uncertain data

## Delivery Rules

- Avoid reading slides verbatim.
- Keep transitions explicit.
- Convert dense bullets into spoken logic.
- Surface uncertainty honestly.
- End key sections with a takeaway.

## Output Standard

Default output order:

1. delivery summary
2. slide-by-slide speaker notes
3. likely Q&A
4. presenter reminders
