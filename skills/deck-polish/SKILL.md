# Deck Polish

Use this skill when a draft deck is mostly correct but needs sharper messaging, cleaner phrasing, and a more senior presentation standard.

## Goals

Improve signal density without changing the underlying facts.

## Polish Workflow

### 1. Tighten messages

- replace topic titles with conclusion titles
- remove filler words
- shorten bullets
- remove duplicate qualifiers

### 2. Raise executive readability

- front-load the conclusion
- keep supporting points parallel
- simplify terminology when possible
- surface risks and asks explicitly

### 3. Reduce clutter

- one message per slide
- three to five bullets max unless the format demands more
- convert long paragraphs into structured points
- split overloaded slides when needed

### 4. Style adaptation

Adapt tone for:

- executive review
- customer presentation
- technical design review
- internal status update

## Output Standard

Default output order:

1. polish summary
2. before-and-after rewrites
3. slide split or merge recommendations
4. final executive-readiness checklist
