# slidemax-clawagent

This repository stores OpenClaw agent workspaces under `agents/`.

Current workspace:

- `agents/ppt/`: PPT-focused OpenClaw workspace for narrative design, deck generation, review, speaker notes, and delivery workflows

Repository-level usage:

```bash
cd agents/ppt
bash scripts/validate_workspace.sh
bash tests/test_workspace_structure.sh
```

Read the workspace guide at `agents/ppt/README.md` before installing or registering the agent.
Read the repository installation runbook at `docs/openclaw-install.md` for OpenClaw setup and registration.
